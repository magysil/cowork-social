## 📝 Cómo hacer tu PR

1. Desarrolla tu API en la carpeta correspondiente
2. Incluye Swagger/Postman collection
3. Deploy en Render o Railway
4. PR con:
   - Link al API deployment
   - Link a Swagger docs
   - README con instrucciones
   - Ejemplos de requests/responses
