## 💡 Cómo Usar Este Roadmap

### Para Contribuidores
1. Revisa las tareas de tu Q actual
2. Elige una tarea no asignada
3. Comenta en el Issue correspondiente
4. Actualiza el checkbox cuando completes

### Para PMs
1. Copia este template a `docs/ROADMAP.md`
2. Personaliza fechas y features según el proyecto
3. Actualiza mensualmente
4. Comunica cambios en README principal

### Herramientas Recomendadas
- **Miro**: Roadmap visual con timeline
- **Notion**: Base de datos de features con propiedades
- **GitHub Projects**: Kanban board con milestones
- **Markdown**: Versionado en Git (este formato)
